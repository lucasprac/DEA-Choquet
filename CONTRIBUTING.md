# Contributing to dea-br

Thank you for your interest in contributing to the `dea-br` library! We welcome contributions from the community to improve the library.

## How to Contribute

### Reporting Bugs
If you find a bug, please create an issue on GitHub with:
- A clear description of the bug.
- Steps to reproduce the issue.
- Expected vs. actual behavior.

### Suggesting Enhancements
If you have an idea for a new feature or improvement, feel free to open an issue or submit a pull request.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your changes.
3. Make your changes and ensure tests pass.
4. Submit a pull request with a detailed description of your changes.

## Development Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install .
   ```
3. Run tests:
   ```bash
   pytest
   ```

## Code Style
Please follow PEP 8 guidelines for Python code.
