# Design System: Japanese Swiss

## Core Philosophy
Minimalist, grid-based, high functional beauty.

## Color Palette
- **Accent**: `#FFF9C4` (Light Yellow)
- **Backgrounds**: Whites (`#FFFFFF`) and very light grays.
- **Text**: Deep grays and blacks for high contrast and readability.

## Typography
- Modern sans-serif (Inter, Roboto, or Outfit).
- Clear hierarchy.

## UI Components
- Sidebar navigation for SaaS feel.
- Card-based dashboard.
- Responsive grid layout.
- Subtle micro-animations (glassmorphism where appropriate).

## Rules
- **NO EMOJIS** in the frontend.
- Clean, premium aesthetics.
